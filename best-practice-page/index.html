"Make it Work, Make it Right, Make it Fast"

Agile web development is a rapid and continuous improvement of software solutions. As what Kent Beck quoted, "Make it work, make it right, make it fast" is one of developer's favorite mantras on agile web development. Thus, agile web development can be summarized into 3 stages.

Make it Work

Ensures that the problem can be solved regardless of the tools used.

Make it Right

If the problem is solvable, improve the code base by following standard principles necessary to have a clean and readable code.

Make it Fast

Optimize the app. Like improving web page load times, etc.
